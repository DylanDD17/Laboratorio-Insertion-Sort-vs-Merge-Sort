# Laboratorio-Insertion-Sort-vs-Merge-Sort

Este codigo contiene la implementación de los algoritmos de ordenamiento Insertion Sort y Merge Sort, utilizados para analizar y comparar su rendimiento al ordenar arreglos de diferentes tamaños.

Los algoritmos fueron implementados con el objetivo de medir sus tiempos de ejecución y estudiar su comportamiento a medida que aumenta el volumen de datos.
